# 开发执行 Prompt

## 项目背景

我要开发一个个人博客兼知识库网站，用于公开展示我的编程学习笔记和技术文章。目标用户包括面试官、同行、朋友和我自己。

网站排版和风格需要高度参考 `https://linzyblog.netlify.app/`，重点是顶部导航、沉浸式首屏、文章卡片流、右侧个人信息栏、分类、标签、归档和清爽卡片化视觉。

具体产品需求请参考 `requirements.md`，详细技术方案请参考 `technical-design.md`，开发规则请参考 `AGENTS.md`。请严格按照这些文档执行，不要自由发挥。

## 当前目标

完成第一版 MVP 静态网站，让用户可以打开首页、浏览文章、搜索文章、按分类和标签筛选文章，并阅读文章详情。

## 第一阶段要完成的功能

- 创建 `index.html`。
- 创建 `assets/css/styles.css`。
- 创建 `assets/js/app.js`。
- 实现首页首屏、导航、文章列表和右侧栏。
- 内置若干编程学习笔记和技术文章示例数据。
- 实现搜索、分类、标签筛选。
- 实现文章详情展示。
- 实现移动端适配。

## 需要创建或修改的文件

- `index.html`
- `assets/css/styles.css`
- `assets/js/app.js`

不要修改无关文件。

## 开发步骤

1. 搭建 HTML 语义结构。
2. 编写参考站风格的 CSS。
3. 编写文章数据和渲染逻辑。
4. 添加搜索、分类、标签筛选。
5. 添加文章详情展示。
6. 做桌面和移动端验证。
7. 修复明显视觉和交互问题。

## 约束条件

- 第一版不做后端。
- 第一版不做数据库。
- 第一版不做登录、评论、点赞、支付。
- 不引入复杂框架。
- 不要把网站做成营销落地页。
- 不要牺牲可读性去堆动画。

## 验收标准

- 打开 `index.html` 能看到完整博客首页。
- 桌面端有首屏、导航、文章列表和右侧栏。
- 移动端没有明显重叠和文字溢出。
- 搜索、分类、标签筛选可用。
- 点击文章可以阅读详情并返回。
- 视觉方向接近参考站：明亮、清爽、圆角卡片、柔和阴影、轻量动效。
